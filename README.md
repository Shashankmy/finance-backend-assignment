# Finance Backend Assignment

## Tech Stack

* Node.js
* Express
* MongoDB Atlas

## Features

* User and role management
* Financial records CRUD
* Dashboard summary APIs
* Role-based access control
* Input validation and error handling

## APIs

* POST /api/users
* GET /api/users
* POST /api/records
* GET /api/records
* GET /api/dashboard/summary

## Setup

1. npm install
2. npm start

## Assumptions

* Authentication is mocked/simplified for local testing purposes
* Role-based access control implemented via middleware

## Database

* MongoDB Atlas (cloud)
